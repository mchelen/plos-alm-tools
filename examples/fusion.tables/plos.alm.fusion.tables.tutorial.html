The PLoS ALM data has been loaded into a set of Google Fusion Tables. The table data can be exported as CSV. It allows filtering, sorting, aggregation, and more with SQL-like queries. The data can for example be loaded in a Google Spreadsheet.
The data is broken into 5 tables: Summary ALM Data, Combined Download Statistics, HTML Views, PDF Downloads, and XML Downloads. The table is referenced with an ID number.

Base URL for Fusion Tables CSV export:
http://tables.googlelabs.com/exporttable?query=
The query should be URL encoded.
Query reference:
https://code.google.com/apis/fusiontables/docs/developers_reference.html#Select

The first table is Summary ALM Data.
Table: Name: PLoS ALM v3 05182010 Summary ALM Data
ID: 204244
http://tables.googlelabs.com/DataSource?snapid=61925

entire table
SELECT * FROM 204244


google docs supports import up to 500kb
unlimited results may not fit on the spreadsheet, so use a limit

Limit results:
SELECT * FROM 204244 LIMIT 10

Select particular columns:
SELECT 'Article Title',DOI,URL FROM 204244 LIMIT 10


Escape special characters:
SELECT '\'Research Article\' or \'non-Research Article\'?' FROM 204244 LIMIT 10

or use column references
SELECT col1,col2 FROM 204244 LIMIT 10

sort results (by citation)
SELECT * FROM 204244 ORDER BY 'Citations - CrossRef' DESC LIMIT 10

Most cited from 2009, filter for articles from 2009 (most crossref citations)
SELECT * WHERE 'Publication Year' = '2009' FROM 204244 ORDER BY 'Citations - CrossRef' DESC LIMIT 10
